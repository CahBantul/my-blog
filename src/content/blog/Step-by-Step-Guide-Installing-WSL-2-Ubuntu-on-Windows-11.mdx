## Step 1: Introduction:

In this tutorial, we'll walk you through the process of installing WSL 2 Ubuntu on your Windows 11 machine. This setup allows you to seamlessly run Ubuntu alongside your Windows environment, enabling you to leverage the power of Linux commands and tools.

## Step 2: Prerequisites:

Before getting started, ensure your system meets the following requirements:

* Windows 11 operating system installed on your PC.
* Virtualization capabilities enabled in your system's BIOS settings.
* Stable internet connection for downloading Ubuntu.

## Step 3: Enabling WSL and Virtual Machine Platform:

First, open PowerShell as an administrator. Execute the following command to enable Windows Subsystem for Linux (WSL):

```shell
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
```

Then, enable the Virtual Machine Platform feature by running:

```shell
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
```

## Step 4: Downloading and Installing Ubuntu:

Head over to the Microsoft Store and search for Ubuntu. Download and install the latest version available.

!Step-by-Step Guide: Installing WSL 2 Ubuntu on Windows 11](../../..../../../src/assets/images/Screenshot_3.png)

## Step 5: Setting WSL 2 as the Default Version:

Open PowerShell as an administrator and execute the following command to set WSL 2 as the default version:

```shell
wsl --set-default-version 2
```

## Step 6: Launching Ubuntu and Initializing Setup:

After installation, launch the Ubuntu application from the Start menu. Follow the on-screen prompts to set up your Ubuntu user account and password.

## Step 7: Updating Ubuntu:

Once Ubuntu is up and running, open the terminal and run the following commands to ensure your system is updated:

```shell
sudo apt update && sudo apt upgrade
```

## Step 8: Customizing Your Ubuntu Environtment:

Explore the Ubuntu terminal and personalize it according to your preferences. Install additional packages, configure settings, and customize your shell to enhance your workflow.

## Step 9: Integration with Windows:

With WSL 2 Ubuntu installed, you can seamlessly interact with Windows files and applications from within the Ubuntu environment, and vice versa, enhancing productivity and flexibility.

## Step 10: Conclusion:

Congratulations! You've successfully installed WSL 2 Ubuntu on your Windows 11 system. Explore the capabilities of both operating systems and enjoy the benefits of this powerful integration.
